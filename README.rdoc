== Multitenant Issues

This is sample project for simple multi-tenant architecture presented on 
Sarajevo 'Ruby Meetup' http://www.meetup.com/ruby/Sarajevo-BA/901742/. 

== How to start

* Clone repository
* bundle install
* rake db:migrate
* rake db:seed
* rails s


There are 2 tenants seeded (db/seed.rb)

* localhost
* acme

To be able to work with acme tanent, add to your /etc/hosts file entry

127.0.0.1   acme


tenants can be accessed via:

* localhost:3000
* acme:3000
