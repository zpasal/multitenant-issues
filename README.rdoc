== Multitenant Issues

This is sample project for simple multi-tenant architecture presented on 
Sarajevo 'Ruby Meetup' http://www.meetup.com/ruby/Sarajevo-BA/901742/. 

== How to start

1. Clone repository
2. bundle install
3. rake db:migrate
4. rake db:seed
5. rails s


There are 2 tenants seeded (db/seed.rb)

- localhost
- acme

To be able to work with acme startup add to your hosts file entri

127.0.0.1   acme


tenants can be accessed via:

localhost:3000

and

acme:3000
